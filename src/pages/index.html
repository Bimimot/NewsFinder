О проекте

Это дипломная работа по итогу обучения на факультете веб-разработки в Яндекс.Практикуме.

Интерфейс адаптирован под различные разрешения экрана, работа со стилями организована по методологии БЭМ. Фронтенд собран вебпаком и бабелем: настроены минификация и префиксер CSS, транспиляция JS.

Для поиска статей используется внешнее API newsapi.org. Собственный REST API для работы бэкенда написан на нативном JS, база данных - NoSQL MongoDB.

Для контроля версионности и прохождения ревью при разработке применялся Git. Бэкенд и фронтенд подняты на одном домене, настроен nginx.
